# Social Preview Brief

GitHub social preview should communicate "article becomes directed video" in one glance.

Starter asset:

- `assets/social-preview.svg`

Export it to PNG before uploading if GitHub does not accept SVG for the social preview image.

## Recommended Composition

Canvas: 1280 x 640.

Left side:

- a markdown/article panel;
- highlighted claim, example, and number;
- label: `Article`.

Center:

- arrows or pipeline steps:
  `Brief -> Spoken Script -> Scene Plan -> QA`.

Right side:

- 3 stacked video frames:
  - comparison chart;
  - process diagram;
  - avatar as PIP over an evidence card.
- label: `Directed Video`.

Footer:

```text
Article to Avatar Video
Chinese long-form content -> visually directed, release-gated avatar videos
```

## Avoid

- full-screen generic avatar face as the only visual;
- provider logos as the hero;
- dark, cinematic, vague AI-video background;
- any real person portrait;
- any competitor name.

## Color And Style

- crisp editor/product screenshot style;
- light background;
- blue/green accent for pipeline status;
- small QA badge: `publish_ready`;
- no fake GitHub stars or unverifiable metrics.
